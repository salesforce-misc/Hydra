== Style

This codebase heavily uses *functional programming* concepts — prefer state transformation over mutation, use monadic operations (`map`, `flatMap`, `fold`) over `when` expressions, and build immutable data flow. See link:{gh-blob}/STYLE.md[STYLE.md] for the full style guide.
